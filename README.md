# DalhousieCapstone_Winter2026
This repository hosts the source code for the Electrical and Computer Engineering (ECE) Capstone Project Bread Rising Monitor at Dalhousie University for the Winter 2026 term.

Some precautions:
  - While uploading the code to ESP32 make sure "Partition Scheme" is set to "No OTA(2MB APP/ 2MB FATFS)". It done through Arduino IDE Tools drop down menu.
  - Make sure SD card is formatted to "FAT32"
  - Arduino IDE
